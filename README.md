# Game Tic tac toe

The game will only work on UNIX systems

# Launch and build

After installation, to build using make and after assembly, you can run ./Tic-tac-toe or make run


# Controls 
```
W/w or KEY\_UP - up 
A/a or KEY\_LEFT - left
S/s or KEY\_DOWN - down
D/d or KEY_RIGHT - right 
enter - to establish
```
